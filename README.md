# Installation

The standard way:

    $ cp creature.vim ~/.vim/colors


The [pathogen][1] way:
    
    $ git clone https://github.com/TaurusOlson/creature.vim.git ~/.vim/bundle/creature.vim


The [vim-plug][2] way:
    
    # In your .vimrc file:
    Plug 'TaurusOlson/creature.vim'
